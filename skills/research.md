# Research Skill

When asked to research any topic, follow this protocol:

## Phase 1: Multi-Source Discovery
1. Search Google Scholar for academic foundations
2. Search Medium for practitioner perspectives  
3. Search Reddit for ground-truth from practitioners
4. Search YouTube for visual explanations
5. Search official documentation for authoritative info

## Phase 2: Synthesis
1. Summarize each source with key claims
2. Identify 3 conflicting viewpoints (if any)
3. Note confidence level of each claim
4. Flag any unverified assertions

## Phase 3: Actionable Output
1. Provide balanced summary with inline citations
2. Suggest 3 actionable takeaways
3. Recommend next steps or deeper reading
4. Estimate time/effort for implementation

## Phase 4: Capture
1. Suggest tags for knowledge base
2. Recommend which notebook/project to store in
3. Offer to generate implementation checklist
