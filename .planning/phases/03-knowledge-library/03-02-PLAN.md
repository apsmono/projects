Pattern 1 (search.py):
```python
async def search_library(
    query: str,
    owner_id: str,
    mode: Literal["keyword", "vector", "hybrid"] = "hybrid",
    limit: int = 12,
) -> list[dict[str, Any]]:
    if mode in ("keyword", "hybrid"):
        keyword_results = _get_store().search_entries(query, limit=limit)
        if mode == "keyword":
            return keyword_results
        if len(keyword_results) >= limit // 2:
            return keyword_results

    # Vector search path
    if not SIGNAL_POSTGRES_DSN:
        return keyword_results if mode == "hybrid" else []

    vec = embed_text(query, task_type="RETRIEVAL_QUERY")
    async with get_pool().connection() as conn:
        rows = await conn.fetchall(
            """
            SELECT entry_id, section, source_url, 1 - (embedding <=> %s::vector) AS similarity
            FROM signal_embeddings
            WHERE owner_id = %s
            ORDER BY embedding <=> %s::vector
            LIMIT %s
            """,
            (vec, owner_id, vec, limit),
        )

    # Enrich with store metadata
    results = []
    for row in rows:
        entry = _get_store().get_entry(row["entry_id"])
        if entry:
            entry["similarity"] = round(row["similarity"], 3)
            results.append(entry)
    return results
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User query -> Vector search | Untrusted text crosses into SQL query |
| Vector DB -> Enriched results | Stale embeddings may leak deleted content metadata |
| Library store -> Search results | Keyword search is in-memory; vector search is DB-backed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04 | Tampering | SQL injection in search query | mitigate | Parameterized queries via psycopg3; query text is passed as embedding parameter, never interpolated into SQL |
| T-03-05 | Information Disclosure | Cross-owner vector search | mitigate | Every query filters by owner_id; no global SELECT |
| T-03-06 | Denial of Service | Vector search fails and breaks library search | mitigate | try/except around vector search call; fallback to keyword results only |
| T-03-07 | Information Disclosure | Stale embedding reveals deleted entry metadata | mitigate | Always join with filesystem store; skip entries where get_entry returns None |
| T-03-08 | Elevation of Privilege | Recent endpoint without auth | mitigate | Both new endpoints use Depends(require_auth) |
</threat_model>
