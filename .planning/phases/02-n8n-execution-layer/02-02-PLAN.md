<objective>
Build the intent executor, credential injection, and template skeleton system — and wire them into the brain's real connect and command paths.

Purpose: This is the core orchestration layer that translates owner intents into n8n workflow executions. It matches a vetted template skeleton, fills its parameters via the Gemini dispatcher (D-01), injects the owner's OAuth credentials into n8n at connect time and before each run so the owner never handles raw keys (D-04/D-05/N8N-02), and gates side-effecting actions through the RL approval system at the owner's actual responsibility level (D-07). The executor is reachable from the brain's command router, and credential sync fires from the sign-in/connect path.

Output: `src/n8n/executor.py` (orchestrator with LLM param fill + runtime credential sync + correct RL gate), `src/n8n/templates.py` (template matching), `src/n8n/credentials.py` (credential sync), 2 workflow skeleton JSONs, governor.py updated with `n8n_workflow` tool entry, `auth_session.py` wired to sync credentials on connect, and `router.py` routing owner intents to `execute_intent`.
</objective>
